Privacy Policy for PlainText

Last updated: January 21, 2026

PlainText is an offline text utility application designed to remove Markdown formatting from text and provide clean plain text output.

This Privacy Policy explains how PlainText handles user data.

1. Data Collection

PlainText does not collect, transmit, store, or process any personal data.

  • No personal information is collected
  • No usage analytics are collected
  • No data is sent to any server
  • No third-party services or SDKs are used

2. Offline Usage

PlainText works entirely offline. All text processing happens locally on the user’s device. The application does not require an internet connection.

3. Text History Storage

PlainText includes an optional feature that stores a history of recently cleaned texts.

  • The history is stored locally on the user’s device only
  • Data is saved using the device’s local storage (SharedPreferences)
  • The data is not shared, uploaded, or accessed externally
  • The user can clear the history at any time

The stored text history never leaves the device.

4. Advertising and Tracking

PlainText:

  • Does not display advertisements
  • Does not use tracking technologies
  • Does not include analytics or marketing SDKs

5. Authentication

PlainText:

  • Does not require user registration
  • Does not support login or authentication
  • Does not create user accounts

6. Third-Party Services

PlainText does not use any third-party services that collect or process user data.

7. Children’s Privacy

PlainText does not knowingly collect any data from children under the age of 13. Since no data collection occurs, there is no risk of personal data exposure.

8. Changes to This Privacy Policy

This Privacy Policy may be updated in the future. Any changes will be reflected on this page with an updated revision date.
